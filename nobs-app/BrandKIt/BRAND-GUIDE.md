# NOBS Brand Guide

Updated June 3, 2026

## Modern Identity

NOBS now uses a simple modern wordmark plus one signal dot.

The logo should feel:

- friendly
- clean
- modern
- secure
- Apple-adjacent
- family-safe

Avoid the old amber tile, shield, lock, robot, or generic AI marks for any new image.
